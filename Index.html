GitHub Sync Setup

Your notes will be saved as a single file in your GitHub repository.

🔑 How to get a token:
Go to github.com/settings/tokens/new → set expiry → tick repo scope → Generate → copy it here.

⚠️ Token is saved only in your browser's local storage, never sent anywhere except GitHub's API.
